# Splash Screen AI Image Generation Prompt

## Main Prompt (for DALL-E, Midjourney, Stable Diffusion, etc.)

```
A modern, vibrant splash screen for a digital drawing and painting application called "Doodle". The design features a clean, minimalist aesthetic with a light gray background. In the center, showcase an artistic arrangement of colorful paint brushes, pencils, and drawing tools arranged in a creative, dynamic composition. Include subtle paint splashes and brush strokes in bright colors (red, blue, green, yellow, cyan, magenta) scattered around the tools. The overall mood should be playful yet professional, conveying creativity and artistic expression. The design should have space in the center or top area for application name text. Style: modern flat design with soft shadows, clean lines, vibrant colors on light gray background, high quality digital art, suitable for software splash screen, 16:9 aspect ratio, professional and inviting.
```

## Alternative Prompt (More Detailed)

```
Create a splash screen design for a digital art application. The background is a soft light gray (#D3D3D3). The main focal point is an elegant arrangement of artistic tools: a paintbrush with colorful bristles, a pencil, and a digital stylus, all rendered in a modern flat design style. Around these tools, add playful elements like colorful paint drops (red, blue, green, yellow, cyan, magenta, orange), small brush stroke marks, and geometric shapes (circles, squares) in various colors. The composition should be balanced and leave space for text overlay. The color palette should be vibrant but not overwhelming. Style: clean modern UI design, soft shadows for depth, professional software splash screen aesthetic, inviting and creative atmosphere, high resolution, suitable for application startup screen.
```

## Simplified Prompt (Shorter Version)

```
Modern splash screen for drawing app "Doodle": light gray background, colorful paint brushes and art tools in center, vibrant paint splashes in rainbow colors, clean flat design style, space for text, professional and creative, 16:9 ratio
```

## Style Variations

### Option 1: Playful & Colorful
Add to any prompt: "playful cartoon style, bright and cheerful, fun artistic elements, whimsical paint splashes"

### Option 2: Professional & Minimal
Add to any prompt: "minimalist design, subtle colors, clean geometric shapes, professional software aesthetic, sophisticated"

### Option 3: Artistic & Creative
Add to any prompt: "artistic brush strokes, abstract paint textures, creative composition, hand-drawn elements mixed with digital design"

## Technical Specifications

- **Aspect Ratio**: 16:9 (recommended for splash screens) or 4:3
- **Resolution**: 1920x1080 or higher
- **Color Scheme**: Light gray background (#D3D3D3 or similar) with vibrant accent colors
- **Text Space**: Leave center or top area clear for "Doodle" text overlay
- **Style**: Modern flat design, clean and professional

## Recommended AI Image Tools

1. **DALL-E 3** (via ChatGPT or Bing)
2. **Midjourney**
3. **Stable Diffusion**
4. **Leonardo.ai**
5. **Adobe Firefly**

## Usage Tips

- Start with the main prompt and iterate based on results
- If the image is too busy, add "minimalist" or "clean design"
- If you want more color, emphasize "vibrant colors" and "rainbow palette"
- For a more professional look, add "corporate software design" or "enterprise application style"
- Specify "no text" if you want to add the application name separately in code
